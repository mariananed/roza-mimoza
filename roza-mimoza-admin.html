Настройки на магазина

🔗 Supabase връзка

Ключовете се задават в файла supabase-config.js:

// supabase-config.js
const SUPABASE_URL = 'https://xxxxx.supabase.co';
const SUPABASE_KEY = 'eyJhbGci...';
  1. Отвори supabase-config.js с Notepad
  2. Замени POSTAVI_TUK_URL с Project URL от Supabase → Settings → API
  3. Замени POSTAVI_TUK_KEY с anon public key
  4. Запази файла (Ctrl+S)
  5. Качи цялата папка отново в Netlify

Паролата се запазва в браузъра на този компютър. При използване на друг компютър или браузър ще важи паролата по подразбиране: RozaMimoza2025
